# todo-app-cli

This app is a CRUD cli tool to manage todos. The todos are stored and managed in a local file.

## Preqrequisites

### Programs, Databases, or Runtime Environments

* `poetry`
* `poethepoet`

### Environment Variables

N/A

## How to Run

### Application

1. `poe run_app`

### Testing, Linting and Formatting

#### Testing

1. `poe test`

#### Linting

1. `poe lint`

#### Formatting

1. `poe format`
